Serial communication

Serial communication using Spring Boot

When data is received from the Meteorological Administration, Linux runs an application made with Spring Boot in the background. Serial operation log generates a loading event by entering information and conditions on the table to be loaded from the table. The serial communication process performs loading on a data having a state value of "0" and an execution state of "N" in the log input to the TBS_JOB_LOG table.

  • Client: Korea Electric Power Corporation
  • Languages/Framworks etc: Spring Boot, gradle, MariaDB etc
